\# HR Analysis Power BI Dashboard



End-to-end Power BI project analyzing HR data with KPIs, dashboards, and business insights.



\## Folder Structure



\- datasets/: Raw CSV/Excel HR data

\- powerbi\_reports/: Power BI (.pbix) files

\- results/: Screenshots of dashboards and KPIs



\## Instructions for Recruiters



1\. Download PBIX file(s) from `powerbi\_reports/`.  

2\. Open in Power BI Desktop to explore interactive dashboards with filters and slicers.  

3\. Screenshots in `results/` show examples of dashboards and KPIs.
